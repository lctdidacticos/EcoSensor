EcoSensor – Estadísticas:
Se trata del desarrollo de una Web App, que genere un Dashboard Interactivo, para utilizar los datos contenidos en un archivo tipo CSV, generado a su vez, por una App para Windows, que complementa el Aparato EcoSensor, dicho archivo contiene Mediciones Ambientales tomadas en Tiempo Real, que incluyen datos de Geolocalización.
Se pretende Hospedar este Dashboard en la WEB, usando los recursos y facilidades que nos brinda la Plataforma RENDER, con la perspectiva posterior de migrar, la actual Aplicación para W+10, hacia esta plataforma, pasando de una App de Escritorio, instalable, a una PWA, que evite procesos de instalación exclusivos para W+10 y así darle acceso Multiplataforma a nuestro proyecto. 
Las funcionalidades deseadas en este Dashboard serian:
1-Generacion de diferentes Graficas contra Tiempo, seleccionando los datos de interés, el intervalo de Datos en el Tiempo y el formato de la gráfica, por el usuario, el archivo CSV ya contiene los Datos debidamente identificados, con el encabezamiento con los nombres de cada Columna.
2- Se debe incluir una Grafica de BARRAS, comparativa que emplea los datos de la Geolocalización, contra los parámetros medidos, ambos seleccionados por el usuario, para realizar evaluaciones comparativas entre diferentes Territorios.
3-El usuario tendrá la opción de DESCARGAR, los Gráficos que seleccione y compartirlo en un formato de Tipo “universal” que permita distribuir estas Graficas, para los fines Académicos de este proyecto.
4-El Dashboard debe ser capaz de realizar una evaluación Estadística de los Datos recibidos y mostrar una sección de Evaluación o Recomendaciones, a partir de estos Datos, opcionalmente se pueden solicitar parámetros de Referencia de indicadores ambientales para comparar con los Datos medidos y emitir Recomendaciones.
